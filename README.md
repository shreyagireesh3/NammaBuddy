 NAMMA Buddy

A smart Bangalore companion app for tourists and newcomers.

Features
- Smart Transport Assistant
- Metro Navigation
- Food Discovery
- Hotel & PG Suggestions
- Emergency Support
- Language Helper
- Voice Translation
- AI Assistant
- Events & Explore
- Must-Try Bangalore Recommendations
 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js

 Purpose
To help tourists and newcomers navigate Bangalore easily from a single platform.
